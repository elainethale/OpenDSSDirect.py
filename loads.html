Loads¶

opendssdirect.Loads.AllNames()¶

The names of all the loads present in the active circuit. The result is delivered as variant, however, the content of this variant is an array of strings.

opendssdirect.Loads.AllocationFactor()¶

The AllocationFactor property of the active load. The parameter argument can be filled with a 0.Set the AllocationFactor property of the active load. The parameter argument must contain the new value in AllocationFactor for the desired active load. The return value will be equal to 0.
